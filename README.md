# Rectangle-area

This repository contains a project to calculate the area of a rectangle.

## Languages Used
- **TypeScript**: 95.8%
- **CSS**: 2.7%
- **JavaScript**: 1.5%

## How to Use
1. Clone the repository.
2. Install the necessary dependencies using `npm install`.
3. Run the project using `npm start`.
